## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```
